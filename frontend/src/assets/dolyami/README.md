# Ассеты для страницы «Оплата Долями»

Используйте **только официальные материалы** Т-Банка / Долями.

## Официальные источники

- **Логотипы (SVG/PNG/EPS):** [tbank.ru/about/brand/](https://www.tbank.ru/about/brand/)
- **Правила по логотипам и баннерам:** [tbank.ru/business/help/sales/loans/about-loans/promo](https://www.tbank.ru/business/help/sales/loans/about-loans/promo)  
  Нельзя менять композицию логотипа, ставить на фотофон, брать фото/3D из макетов и т.п.
- **Маркетинговые материалы Долями:** [dolyame.ru/brand/](https://dolyame.ru/brand/)
- **Обязательные дисклеймеры для партнёров:** из guidebook Т-Банка (см. [tbank.ru/business/loans/integration/guidebook](https://www.tbank.ru/business/loans/integration/guidebook/) и материалы для партнёров).

## Логотип в проекте

В проекте используется плейсхолдер `public/images/dolyami/logo.svg`. Официальный логотип скачайте с [tbank.ru/about/brand/](https://www.tbank.ru/about/brand/) или [dolyame.ru/brand/](https://dolyame.ru/brand/) и замените файл по пути `public/images/dolyami/logo.svg`. Информер и страница `/dolyami` подхватят его автоматически.
