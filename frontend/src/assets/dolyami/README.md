# Ассеты для страницы «Оплата Долями»

Используйте **только официальные материалы** Т-Банка / Долями.

- Маркетинговые материалы и гайд по фирменному стилю: [dolyame.ru/brand/](https://dolyame.ru/brand/)
- Промоматериалы для партнёров: [tbank.ru/business/help/sales/loans/about-loans/promo](https://tbank.ru/business/help/sales/loans/about-loans/promo)

**Логотип:** в проекте используется плейсхолдер `public/images/dolyami/logo.svg`. Чтобы подключить официальный логотип, скачайте его с [dolyame.ru/brand/](https://dolyame.ru/brand/) (Figma или PDF-гайд) и замените файл по пути `public/images/dolyami/logo.svg` (или обновите константу `DOLYAMI_LOGO` в компонентах). Информер на главной и страница `/dolyami` подхватят новый файл автоматически.
